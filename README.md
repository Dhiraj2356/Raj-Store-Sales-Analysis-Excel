# Store Sales Analysis using Excel

##  Project Overview
This project focuses on analyzing store sales data using Microsoft Excel. The objective is to transform raw sales data into meaningful insights through data cleaning, structuring, and an interactive dashboard.

The project demonstrates practical Excel skills commonly used in sales reporting and business analysis.

---

##  Objectives
- Clean and structure raw sales data for analysis  
- Identify sales trends and performance metrics  
- Build an interactive dashboard for easy decision-making  

---

##  Dataset
- Source: Store sales dataset (provided in Excel format)
- Data includes sales details such as product, category, State, quantity, revenue etc.

---

##  Data Cleaning & Preparation
The following steps were performed in Excel:
- Checked and handled missing (null) values  
- Structured uneven and inconsistent data  
- Standardized data formats for accuracy  
- Created additional calculated columns for better clarity and analysis  

---

##  Dashboard & Analysis
- Created multiple Pivot Tables for sales analysis  
- Built 6 Pivot Charts to visualize key metrics  
- Combined all charts into a single dashboard sheet  
- Implemented slicers connected across all Pivot Tables for dynamic filtering  

The dashboard allows analysis by different dimensions such as product, category, region, and time.

---

## Insights & Findings
- Women are more likely to buy as compare to Male (64% Women)  
- Maharashtra Karnataka are the top three States   
- Adult Age group (21-49) is Max Contributing
- Most purchases are concentrated in sizes S, M, and L across products
- Amazon, Flipkart and Myntra are max contributing channels

These insights can help business stakeholders focus on high-performing states , channels, and improve regional sales strategies.
